## 로드맵 (지금 안 함 — "예정" 밖)

| 구분 | 기능 | 비고 |
|------|------|------|
| 하네스 | 조건분기 · 서브에이전트 · 병렬 | 엔진 복잡도↑ |
| 트리거 | 무인 실행(꺼져 있어도) | 클라우드 워커+Celery+Redis+vault |
| 관측 | 실행 로그(run_log)·트레이스·비용 | Grafana/Prometheus/Sentry |
| 동기화 | 실시간 파일 감시 | 지금은 "다시 읽기" 버튼 |
| 갤러리 | 검색 · 좋아요 | 검색 제외, 좋아요→가져가기수 대체 |
